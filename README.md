# User Profile Manager 👤

A simple JavaScript project that demonstrates how to work with **Objects**.
The project simulates a basic **user profile system** where we can display, update, add, delete, and check properties inside a JavaScript object.

This project is part of my JavaScript learning journey and focuses on understanding how objects store and manage related data.

---

## 📌 Features

* Display user information
* Access specific properties from an object
* Update existing object properties
* Add new properties dynamically
* Delete properties from an object
* Check if a property exists using the **in operator**

---

## 🧠 Concepts Used

This project practices important JavaScript fundamentals:

* JavaScript Objects
* Object Properties
* Dot Notation
* Bracket Notation
* Updating Object Values
* Adding New Properties
* Deleting Properties
* Conditional Statements
